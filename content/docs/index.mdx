# Glyphic Documentation

Glyphic turns plain-text Mermaid diagrams into living, interactive visuals — with animation, status overlays, presentation labels, and before/after comparison built in.

Glyphic extends Mermaid with meta-blocks — directives appended after the diagram definition.

  <Card title="@animate" href="/docs/meta-blocks/animate" description="Animate edges with flowing particles and nodes with pulsing borders." />
  <Card title="@status" href="/docs/meta-blocks/status" description="Live NOC board — map nodes to healthy, degraded, or down states." />
