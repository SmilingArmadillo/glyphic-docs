# Welcome to Glyphic

Glyphic turns plain-text Mermaid diagrams into living, interactive visuals. Whether you're documenting a microservices architecture, walking through a deploy pipeline, or presenting a system design in a team meeting, Glyphic gives you the tools to make diagrams that move, compare, and tell stories.

## What you can do with Glyphic

- **Edit** Mermaid source in a live editor with instant preview
- **Animate** edges and nodes with the `@animate` meta-block
- **Compare** before/after states with `@compare` split view
- **Export** diagrams as SVG, PNG, GIF, or WebM
- **Share** a view-only link so teammates see the diagram without touching the source
