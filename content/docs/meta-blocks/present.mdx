# @present

The `@present` block adds visual labels to the diagram: badges attached to nodes and fixed overlays positioned on the canvas.

```text
@present
badge {nodeId}: "{text}" [type={type}] [color={color}] [position={position}]
overlay {anchor}: "{text}" [color={color}] position=fixed
```

<Tabs items={["badge", "overlay"]}>
  <Tab value="badge">

## `badge`

A pill-shaped label rendered in the SVG, attached to a node and moving with it.

```text
@present
badge Server: "⚠ Overloaded" type=error
badge LB: "Load Balancer" type=info
badge DB: "Healthy" type=success color=#10b981
```

## `overlay`

A text label positioned at a fixed viewport location. Useful for metric callouts, status text, or diagram annotations.

```text
@present
overlay top-right: "Queue: 500+" position=fixed color=#ef4444
overlay bottom-left: "p99: 2.3s" position=fixed color=#ef4444
```
