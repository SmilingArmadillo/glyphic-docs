Meta-blocks are Glyphic-specific extensions appended **after** the diagram definition. They use a simple keyword syntax:

```text
flowchart LR
  A --> B --> C

@animate
A-->B: type=line-flow, color=#38bdf8, speed=2

@present
badge B: "Processing" type=info

@compare split

flowchart LR
  A --> B

@compare
title before: "Before"
title after: "After"
```

<Callout type="info">
**Mermaid compatible.** Strip everything from the first `@` line downward and you have valid, portable Mermaid that renders in any standard renderer. Meta-blocks are purely additive.
</Callout>

## Syntax rules

- Meta-blocks follow the diagram definition — never before it
- Each block opens with `@keyword` on its own line
- Entries continue until the next `@keyword` or end of file
- Meta-blocks are case-sensitive
- Multiple meta-blocks can appear in the same file
