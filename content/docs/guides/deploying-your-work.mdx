# Deploying Your Work

## Docker (recommended)

```bash
docker compose up -d
```

Use this `docker-compose.yml`:

```yaml
version: '3.9'
services:
  frontend:
    image: ghcr.io/smilingarmadillo/glyphic-frontend:latest
    ports: ['3000:3000']
    environment:
      VITE_API_BASE: http://api:3001
  api:
    image: ghcr.io/smilingarmadillo/glyphic-api:latest
    ports: ['3001:3001']
    environment:
      REDIS_URL: redis://redis:6379
  redis:
    image: redis:7-alpine
    volumes: [redis-data:/data]
volumes:
  redis-data:
```

## Reverse proxy

Route `/docs` to the docs Next.js app and everything else to the Vite frontend.

| Path | Target |
|---|---|
| `/docs/*` | `glyphic-docs:3002` |
| `/api/*` | `glyphic-api:3001` |
| `/*` | `glyphic-frontend:3000` |

## Environment variables

| Variable | Service | Description |
|---|---|---|
| `REDIS_URL` | API | Redis connection for share storage |
| `VITE_API_BASE` | Frontend | API base URL |
| `PORT` | Docs | Port for the Next.js docs server |
