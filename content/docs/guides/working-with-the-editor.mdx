# Working with the Editor

## Layout

The editor is a two-column layout: source on the left, live preview on the right. The toolbar runs across the top.

## Keyboard shortcuts

| Shortcut | Action |
|---|---|
| `⌘S` / `Ctrl+S` | Save (no-op in browser, triggers export prompt in app) |
| `⌘K` | Open search |
| `⌘Z` | Undo |
| `⌘⇧Z` | Redo |
| `⌘+` | Zoom in |
| `⌘-` | Zoom out |
| `⌘0` | Reset zoom |

## File upload

Drag a `.mmd` or `.txt` file onto the editor to load it. Click the upload area to browse.
