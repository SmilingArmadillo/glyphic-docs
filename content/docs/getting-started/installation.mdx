# Installation

## Use the hosted version

No installation required. Open [glyphic.cc/app](https://glyphic.cc/app) in your browser and start editing immediately.

## Self-host with Docker

```bash
docker pull ghcr.io/smilingarmadillo/glyphic:latest
docker run -p 3000:3000 ghcr.io/smilingarmadillo/glyphic:latest
```

Visit `http://localhost:3000` to open the editor.

## Self-host from source

```bash
git clone https://github.com/SmilingArmadillo/mermaid-studio.git
cd mermaid-studio
pnpm install
pnpm dev
```

## Environment variables

| Variable | Required | Description |
|---|---|---|
| `REDIS_URL` | Yes (sharing) | Redis connection string for share storage |
| `API_PORT` | No | Port for the Node.js API server (default: 3001) |
| `VITE_API_BASE` | No | Base URL the frontend uses to reach the API |
