# Quickstart

<Steps>
  <Step>

## Open the editor

Go to [glyphic.cc/app](https://glyphic.cc/app). The editor opens with an example diagram pre-loaded in the left panel and a live preview on the right.

  </Step>
  <Step>

## Write a diagram

Clear the editor and paste:

```text
flowchart LR
  User[User] --> LB[Load Balancer]
  LB --> S1[Server 1]
  LB --> S2[Server 2]
  S1 --> DB[(Database)]
  S2 --> DB
```

The preview updates instantly.

  </Step>
  <Step>

## Add an animation

Append this block below your diagram (after the last line, no blank line required):

```text
@animate
User-->LB: type=line-flow, color=#38bdf8, speed=2
LB-->S1: type=line-flow, color=#22c55e, speed=2.5
LB-->S2: type=line-flow, color=#22c55e, speed=2.5
```

The edges begin animating in the preview immediately.

  </Step>
  <Step>

## Export

Click **Export** in the toolbar and choose a format:

| Format | Best for |
|--------|----------|
| SVG | Docs, READMEs, embedding |
| PNG | Slide decks, PR comments |
| GIF | Animated, universal support |
| WebM | High-quality video, Notion embeds |

  </Step>
</Steps>
