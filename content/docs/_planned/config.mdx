# @config

<Callout type="warn">
**Not yet implemented.** The syntax below is reserved — do not ship diagrams relying on it. This page exists to document planned behaviour before implementation.
</Callout>

The `@config` block sets global diagram defaults: font, primary colors, animation defaults, and Mermaid base theme. Per-edge and per-node `@animate` entries always override global defaults.

## Font

```text
@config
font: IBM Plex Sans
```

## Global animation defaults

Sets baseline defaults for all animated edges. Per-edge `@animate` entries override these for individual edges.

```text
@config
animation.dot.color: #f59e0b
animation.dot.speed: 1.2
animation.lineFlow.color: #3b82f6
animation.lineFlow.speed: 2
animation.lineFlow.motionPreset: balanced
animation.lineFlow.glowPreset: none
```
