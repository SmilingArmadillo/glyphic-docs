# Authentication

## Getting an API key

API keys are available on the [Dashboard](https://glyphic.cc/dashboard) under **Settings → API Keys**.

## Using your key

Include the key as a bearer token in every request:

```bash
curl https://glyphic.cc/api/shares \
  -H "Authorization: Bearer glyph_live_abc123" \
  -H "Content-Type: application/json" \
  -d '{"source": "flowchart TD\n  A --> B", "background": "dark", "type": "editable"}'
```

## Key types

| Type | Prefix | Permissions |
|---|---|---|
| Live | `glyph_live_` | Full API access |
| Read-only | `glyph_ro_` | GET endpoints only |
| Test | `glyph_test_` | Sandboxed, no persistence |

## Error codes

| Code | Meaning |
|---|---|
| `401` | Missing or invalid token |
| `403` | Token lacks required permission |
| `429` | Rate limit exceeded |
