# Rendering API

## Render Mermaid source

```
POST /api/render
```

Renders Mermaid source server-side and returns an SVG string. Useful for CI pipelines, documentation generators, and preview thumbnails.

**Request:**

```json
{
  "source": "flowchart TD\n  A --> B",
  "background": "dark",
  "theme": "default"
}
```

**Response:**

```json
{
  "svg": "<svg xmlns='http://www.w3.org/2000/svg'>...</svg>",
  "width": 400,
  "height": 300
}
```

## Retrieve view-only SVG

```
GET /api/shares/:id/render
```

Returns the rendered SVG for a view-only share. The Mermaid source is never included in the response.

| Field | Type | Description |
|---|---|---|
| `svg` | string | Rendered SVG markup |
| `background` | `dark` or `white` | Background the diagram was saved with |
